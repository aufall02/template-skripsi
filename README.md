PANDUAN PENGGUNAAN TEMPLATE

Versi: 0.9

Deskripsi

Template_Skripsi.dotm adalah template Microsoft Word untuk penulisan
skripsi dengan Style, TOC, dan Macro.

Fitur

-   Style terstruktur
-   Heading 1, Heading 2, Heading 3
-   FrontMatter
-   Caption
-   Daftar Isi otomatis
-   Cross Reference
-   Navigation Pane
-   Macro Caption Gambar & Tabel

Cara Menggunakan

1.  Simpan Template_Skripsi.dotm.
2.  Double-click file tersebut.
3.  Klik Enable Content jika diminta.
4.  Word membuat dokumen baru.
5.  Simpan dokumen baru dengan nama skripsi Anda.

Style

-   Title : Cover
-   FrontMatter : Halaman awal
-   Heading 1 : BAB
-   Heading 2 : Subbab
-   Heading 3 : Sub-subbab
-   Normal : Isi
-   Caption : Judul gambar/tabel

Shortcut

-   Ctrl+Shift+G : Caption Gambar
-   Ctrl+Shift+T : Caption Tabel
-   Alt+Ctrl+1	 : Heading 1
-   Alt+Ctrl+2	 : Heading 2
-   Alt+Ctrl+3	 : Heading 3
-   Ctrl + A  F9 : Update Seluruh field


Aturan

-   Jangan edit template .dotm secara langsung.
-   Gunakan style yang tersedia.
-   Jangan mengetik nomor heading manual.
-   Jangan mengubah font manual.

Troubleshooting

-   Macro tidak jalan: klik Enable Content.
-   TOC tidak update: Ctrl+A lalu F9.
-   Caption salah nomor: Ctrl+A lalu F9.
